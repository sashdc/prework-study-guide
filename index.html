HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Tags must open and close to ensure content is managed by them.
  • One H1 header per page.
  • The body can contain paragraphs, images, and lists.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector before the declaration/rule says which parts to apply the declaration or rule to
  • * as a selector aplies it to the whole page
  • .card as a selector applies it to all elements in the page that are marked as cards
  • You can apply one style to multiple elements by separating them with a comma in the selector

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A : adds changed files in working directory to staging area of GIT
  • git commit -m : associates a message with our commit to know what changes were made
